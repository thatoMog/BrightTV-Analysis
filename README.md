# BrightTV-Analysis
📌 Project Overview
BrightTV’s CEO has set a strategic objective to grow the company’s subscription base for the current financial year. This case study leverages user profile data and viewer transaction records to generate actionable insights for the Customer Value Management (CVM) team.

🎯 Objectives
Provide insights on user and usage trends.

Identify factors influencing consumption.

Recommend content strategies to boost engagement on low-consumption days.

Suggest initiatives to grow BrightTV’s user base.
